# Synapse Agent Framework: Enterprise-Grade Autonomous Orchestration

**Synapse Agent Framework** は、Google Cloud Vertex AI の能力を最大限に引き出し、エンタープライズ環境で「自律型マルチエージェント」を構築・運用するための TypeScript フレームワークです。

従来のチャットボットを超え、複雑な業務フローを AI が自律的に判断し、状態を管理しながら完結させる「AI ワーカ」を、圧倒的な低コストと堅牢性で実現します。

---

## 🚀 主要なベネフィット

### 1. 「伝言ゲーム」による情報変質の完全排除
複数の AI エージェントを連携させる際、従来の「要約されたメッセージの受け渡し」では、AI による勝手な解釈や重要な ID・コンテキストの欠落（変質）が避けられませんでした。Synapse は、全エージェントが物理的に同一の共有メモリ（GlobalState）を参照。親から子へは「何をしてほしいか」の最小限の JSON 指示のみを送り、前提知識はメモリから直接読み取らせることで、情報の劣化をゼロにします。

### 2. プロンプト・インジェクションに対する堅牢な防御
ユーザーの生の入力を受け取る「フロントエージェント」と、実際の処理を行う「ワーカージェント」を完全に分離。悪意のある入力の無害化を入り口で一括管理し、後続のエージェントには安全に構造化された JSON データのみが渡されるため、システム深部へのプロンプト攻撃を構造的に遮断します。

### 3. ステート圧縮によるトークンコストの劇的な削減
会話履歴をすべて LLM に再送する従来の手法とは異なり、Synapse は現在の状況を「State（記憶）」として構造化し、必要なエッセンスのみをインプットに含めます。これにより、セッションが長引いても入力トークンが肥大化せず、常に高い推論精度と圧倒的な低コストを維持します。

### 4. コストパフォーマンスの最大化（モデルスイッチング）
タスクの分解や計画立案を行う最初のステップだけを高性能な `Gemini-2.5Pro` で行い、実際のデータ処理（ツール実行）ステップは高速・安価な `Gemini 2.5 Flash` で回すといった、ステップ単位での動的なモデル切り替えを標準サポートしています。

---

## 🛠️ 技術的な特徴

- **Vertex AI / Gemini API 専用設計**: Google Cloud への統合が容易で、高速なレスポンスと高いセキュリティを保証。
- **セマンティック・ガードレール**: Harness により、AI の思考プロセスにリアルタイムで介入。機密情報の出力ブロックや、RAG 経由のコンテキスト注入を柔軟に実装可能。
- **厳格なスキーマ制御**: 入出力に JSON スキーマを強制。AI の解釈揺れを排除し、既存の基幹システムとカッチリと連携。
- **難読化・知財保護**: コアロジックを難読化した状態で提供。独自のプロンプトノウハウをリバースエンジニアリングから守ります。

## 💳 ライセンスと導入フロー

Synapse は、開発者の皆様がリスクなく導入を開始できるよう「フリーミアム・モデル」を採用しています。

1.  **無料で開始 (Free Tier)**:
    - **4エージェントまでの構成**であれば、ライセンスキーなしですべての機能を無料でご利用いただけます。PoCやプロトタイプ開発に最適です。
2.  **Marketplace で購入 (Pro/Enterprise)**:
    - 5エージェント以上の大規模マルチエージェント構成、または商用サポートが必要な場合は、Google Cloud Marketplace からサブスクリプションを登録してください。
3.  **ライセンスキーの発行**:
    - 購入後、管理コンソールから `syn_` で始まるライセンスキーが発行されます。
4.  **NPM から導入**: 
    ```bash
    npm install @synapse-agent/framework
    ```
